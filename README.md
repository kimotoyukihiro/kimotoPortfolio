# <美容商材でもっと綺麗に！>

## サイト概要
美容室専売品のレビューサイト

### サイトテーマ
美容室専売品を販売するために、美容師さんが商品のレビューを書き込むサイト

### テーマを選んだ理由
前職で美容室に営業に回っていた際に、商品が売れないことが課題だったため

### ターゲットユーザ
美容師さんとそのお客さん

### 主な利用シーン
美容室の宣伝及び、美容商材の紹介
